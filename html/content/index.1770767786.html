How to use

Submitting scores

Paste the game match scores into the "results" field.

Ensure the scores are formatted correctly. See: Table formatting.

Viewing Results

Once you've submitted the scores, an image displaying scores and results of the match will be shown.

Scores and results will be updated in real-time as you type.

Boards

You can switch between "Ranked" and "Unranked" matches. using a switch button under the "results" field.

  1. Ranked board - match results will use the ranked leaderboards and players' ratings for calculation.
  2. Unranked board - match results won't be calculated using the ranked leaderboards. Ideal for custom matches.

Copying Scores

Under the "results" field, you'll also find two copy buttons.

  1. Copy - copies the "results" field value as is to your clipboard. - might not be compatible with Discord's !table command
  2. #lobby - copies the "results" in a format compatible with Discord's !table command. Use this button to copy the "results" field value to your clipboard and then paste it into a Discord #lobby-room.

Using Presets

The "Presets" dropdown provides a quick way to load predefined templates for different ranked lobby modes.

  1. Click on the "Presets" dropdown to view the available options.
  2. Select a lobby mode from the list, and the preset will be loaded into the "results" field.

Lobby modes

Downloading Results

Under the generated image, you'll find a download button, which downloads the results image to your device.

Match History and Comparison

If "Ranked" board is selected and the lobby number in the results field matches a previously submitted match for that lobby mode, the submitted match can be viewed.

A message will appear: "Lobby #number submitted".

Click on the "Show" button to temporarily load the scores and results of this lobby instead of the current match. Your match results will stay loaded in the "results" field.

To switch back, click on the "Hide" button.

Automatic board updates

The leaderboards automatically update and refresh at regular intervals.

These updates occur every few minutes.

As a result, there might be a slight delay in displaying the most recent matches and player rankings.

URL Parameters

Some settings can be configured through URL parameters.

TABLE

Preload scores (ensure the template is URL encoded).

The scores will be automatically filled in the "scores" field when the website is loaded.

Example:

https://ctrlobbies.com/?table=Lobby%20%23999999%20-%204%20vs.%204...

BOARD

Set "Ranked" or "Unranked" board.

Example:

https://ctrlobbies.com/?board=ranked
https://ctrlobbies.com/?board=unranked

- "board=ranked" is a default option

Parameters can be combined:

https://ctrlobbies.com/?board=unranked&table=Lobby%20%23999999%20-%204%20vs.%204...

Editing Scores

Opening the Editor

Click the "Edit scores" button at the top of the page, or click on the generated image to open the editor.

Selecting a Race

Choose the race for which you want to add scores from the dropdown.

"Race #1", "Race #2", etc. correspond to each race in the lobby.

Select "Auto" to automatically choose the first race where no points have been assigned yet to any player.

Assigning Race Positions

Drag and drop players in the list to set their finishing positions.

Click the "Apply" button to save the scores. Points are assigned automatically.

Applying Penalties

Click the "[P]" button (on the right) to select a penalty or zero points to a player.

Tying Players

Click a player's position number (on the left) to tie or untie them with the player above.

Applying changes

Click on the "Apply" button to save scores for the currently selected race.

Each race must be applied individually. Changes are applied only to the currently selected race.

Table formatting

Header

Header is required for all matches.

Format:

Lobby #number - lobby mode

Lobby number is optional. If it is missing then a default lobby number will be used.

Lobby mode can be spearated by a space or a dash.

Examples:

Lobby #1000 - FFA
Lobby #3682 - Itemless Duos
Lobby #21 - Battle 3 vs. 3
Lobby - Insta 4 vs. 4
Lobby FFA

You can load in "Presets" to see valid lobby headers. See: How to use.


Itemless lobbies use Elo calculation.

Item lobbies use MMR calculation.

Teams and players

Solo modes

Player_1 [flag_code] 0|0|0|0|0|0|0|0
Player_2 [flag_code] 0|0|0|0|0|0|0|0
...

Team modes

Team A #color
Player_1 [flag_code] 0|0|0|0|0|0|0|0
Player_2...

Team B - description #color
Player_3 [flag_code] 0|0|0|0|0|0|0|0
Player_4...

- duplicate player names are not allowed- team name descriptions will be displayed below the team name

Penalty and Bonus [Optional]

Every positive number is a bonus and every negative number is a penalty.

For a player

Player penalty or a bonus can be specified using a negative number as a part of any race:

Player_1 [flag_code] 0|0|0|0|0|9-4|0|0|0
Player_2 [flag_code] 0|0|0|0|0|0|0|0|-10
Player_3 [flag_code] 0|0|0|0|0|0|0|0|0-4
Player_4 [flag_code] 0|-6|0|0|0|0|0|0|0

For a team

Team penalty or a bonus can be specified on a new line after team players via keyword "Penalty" or "Bonus":

Team A #color
Player_1 [flag_code] 0|0|0|0|0|0|0|0
Player_2...
Penalty -10
Bonus 2

No flag [Optional]

Player_1 0|0|0|0|0|0|0|0

Tracks [Optional]

Format:

Track names can be listed one per line after the players.


You can enter full names or approximate names — the system will try to auto-match them.

Examples:

...
Player_7 [flag_code] 0|0|0|0|0|0|0|0|0
Player_8 [flag_code] 0|0|0|0|0|0|0|0|0

Dragon Mines
Clockwork Wumpa
Cortex Castle
N. Gin Labs
Roo's Tubes
Hot Air Skyway


After listing all tracks, the system will parse them and match them to known tracks automatically.

You can enter full names, approximate names, or acronyms — the system will try to auto-match them.

For example, 'Roos', 'Cove', 'Ngin', or acronyms like 'DTD', 'HAS', 'dsd' are recognized.

See "Presets" for an example.

Track can be selected in the scores editor.

How to use the score editor: View

Unranked board ratings [Optional]

The "Ranked" mode uses players' leaderboard ratings.

If you choose "Unranked" mode a default rating will be set for every player.

You can override players' board ratings by specifying a rating number with an "r" prefix after the scores.

Examples:

Player_1 [flag_code] 0|0|0|0|0|0|0|0 r1354
Player_2 [flag_code] 0|0|0|0|0|0|0|0 -100 r1600


If you set a rating number while in "Ranked" mode, the change will not take effect!

Comments [Optional]

If you need to add comments to the scores, you can use the "//" or "#" sign at the start of the line.

Comments won't affect the table in any way.

Examples:

// this is a comment

Lobby #999999 - FFA

Player_1 0|0|0|0|0|0|0|0
# this is another comment
Player_2 0|0|0|0|0|0|0|0
...
